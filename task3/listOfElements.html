1. The <a> tag defines a hyperlink, which is used to link from one page to another.
  2. The <em> tag denotes stressed importance.
  3. The <strong> tag represents strong importance, seriousness, or urgency for its contents.
  4. The <small> tag represents side comments such as small print.
  5. The <s> tag represents contents that are no longer accurate or no longer relevant.
  6. The <cite> tag represents a reference to a creative work.
  7. The <q> tag represents some phrasing content quoted from another source.
  8. The <dfn> tag represents the defining instance of a term.
  9. The <abbr> tag represents an abbr or acronym, optionally with its expansion.
  10. The <data> tag represents its contents, along with a machine-readable form of those contents in the value attribute.
  11. The <time> tag defines a human-readable date/time.
  12. The <code> tag represents a fragment of computer code.
  13. The <var> is a phrase tag. It defines a variable.
  14. The <samp> is a phrase tag. It defines sample output from a computer program.
  15. The <kbd> is a phrase tag. It defines keyboard input.
  16. The <sup> element represents a superscript and the <sub> element represents a subscript.
  17. The <i> tag defines a part of text in an alternate voice or mood, displayed in italic.
  18. The <b> tag specifies bold text.
  19. The <u> tag represents some text that should be stylistically different from normal text.
  20. The <mark> tag defines marked text.
  21. The <ruby> tag specifies a ruby annotation.
  22. The <rp> tag can be used to provide parentheses around a ruby text, to be shown by browsers that do not support ruby annotations.
  23. The <bdi> tag represent span of text that is to be isolated from its surroundings for the purposes of bidirectional text formatting.
  24. The <bdo> tag is used to override the current text direction."text"
  25. The <span> tag is used to group inline-elements in a document.
  26. The <br> tag inserts a single line break
  27. The <wbr> tag specifies where in a text it would be ok to add a line-break.
